Приложение для отображения погоды:

- Позволяет узнавать текущую погоду

- Поиск по городу и по координатам

- Приложение обрабатывает единовременно только один запрос, остальные становятся в очередь и выполняются друг за другом. Пользователь информируется о задержке и автоматический перекидывается к результатам поиска, когда до него дойдет очередь.
